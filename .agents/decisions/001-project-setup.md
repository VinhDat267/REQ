# Decision 001: Chọn đề tài Wonton POS

## Ngày: 03/03/2026
## Trạng thái: Đã quyết định

## Bối cảnh
Cần chọn đề tài cho dự án môn Software Requirement Specification (REQ). Yêu cầu tìm client thực tế, xác định vấn đề, đề xuất giải pháp.

## Quyết định
Chọn **Hệ thống POS cho quán mì vằn thắn** vì:
- Client thực tế (chủ quán mì)
- Vấn đề rõ ràng: quản lý thủ công, sai sót đơn hàng
- Scope phù hợp cho dự án 15 tuần
- Nhiều use cases thú vị (order, payment, kitchen, analytics)

## Hệ quả
- Cần phỏng vấn chủ quán để thu thập requirements
- Nhóm 8 người, mỗi người chọn 2 UC riêng biệt
- Hệ thống gồm 2 WebApp: Client + Admin
